# Respiratory Audio Classification

This project implements an audio preprocessing and deep learning pipeline for respiratory sound, lung sound, and emotional speech datasets.

## Work Completed
- Audio preprocessing
- Normalization
- Silence removal
- MFCC feature extraction
- Feature dataset creation
- Label encoding
- Train-test split
- Deep learning model training using TensorFlow/Keras

## Technologies Used
- Python
- Google Colab
- Librosa
- TensorFlow
- NumPy
- Pandas
- Scikit-learn
